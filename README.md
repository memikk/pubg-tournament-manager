# PUBG Tournament Manager

A desktop utility for organizing PUBG tournaments, entering match results, and exporting leaderboard-style summaries.

## Overview

PUBG Tournament Manager was built to simplify repetitive tournament administration tasks. Instead of managing rankings manually, the application provides a structured desktop interface for match input, team setup, and result tracking.

## Highlights

- team setup workflow for tournament creation
- match-by-match result entry
- leaderboard-oriented data presentation
- export-friendly tournament records
- desktop UI built with Tkinter

## Tech Stack

- Python
- Tkinter
- CSV
